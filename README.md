#Base Ink
Grunt-driven email templating using Zurb Ink responsive email framework.

####Use
Make sure you have node.js installed.

Then run the following in the terminal/shell:
```
git clone git@github.com:wadehammes/base-ink.git
cd base-ink
npm install
grunt
```
